# Shortcuts in Windows

Note: `❖` means Windows key.

## Useful shortcuts

| Shortcut | Action |
| --- | --- |
| `❖` + `D` | Show Desktop |
| `❖ + D` | Show Desktop |
| `❖ D` | Show Desktop |
